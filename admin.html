Administration Solid'R

Cette page est pensée pour une gestion simple sans code. Elle reste légère : pour une vraie sécurité serveur, il faudra ensuite passer par GitHub + CMS ou un backend. Ici, l’accès est protégé par un mot de passe local pour éviter un accès libre à n’importe qui.

Mot de passe temporaire de démonstration : SolidR-admin-2026. Change-le avant publication.

Mode d’emploi

1. Modifie les champs. 2. Clique sur “Générer le JSON”. 3. Télécharge le fichier. 4. Remplace content.json dans le site GitHub Pages. Pour les nouvelles images, ajoute-les dans le dossier img puis indique le chemin, par exemple ./img/nouvelle-action.jpg.
